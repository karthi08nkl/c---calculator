# Calculator in C

A beginner-friendly **Calculator** built in **C** that performs basic arithmetic operations on multiple user-entered numbers. The program uses dynamic memory allocation to store user input and supports addition, subtraction, multiplication, and division through a command-line interface.

## Features

* Accepts multiple numbers as input
* Performs addition
* Performs subtraction
* Performs multiplication
* Performs division
* Uses dynamic memory allocation (`malloc`)
* Handles division by zero
* Interactive command-line interface

## Technologies Used

* C Programming Language

## Concepts Used

* Variables and Data Types
* Loops (`for`)
* Conditional Statements (`if`, `switch`)
* Arrays
* Dynamic Memory Allocation (`malloc` and `free`)
* User Input and Output (`scanf`, `printf`)
* Arithmetic Operations


Created as a C programming practice project.
